This web-based scouting data visualizer was developed for the 2025 FIRST Robotics Competition: Reefscape and tailored to the needs of FRC Team 226, the Hammerheads. The system can be easily customized or adapted for other teams, allowing users to prioritize and display the metrics most relevant to their specific strategy. The application is hosted on Vercel for seamless deployment and offline functionality.

Vercel Link: https://frc-226-reefscape-visualizer-final.vercel.app/

Key Features

* CSV uploads for match data, match schedules, and pit-scouting information
* Team rankings by EPA, Coral Scoring, and Algae Scoring metrics
* Ranking dashboard to compare team averages, filter statistics, and analyze performance trends
* Performance graphs showing team scoring consistency and reliability
* Team comparison tools to evaluate multiple teams side by side
* Sorting features to rank teams based on robot capabilities or design criteria
* Match prediction tool for simulating gameplay outcomes with detailed projections
* Scouting schedule generator for event planning and organization
* Data verification and updates using The Blue Alliance (TBA) APIs
* Full offline functionality for use without internet access
